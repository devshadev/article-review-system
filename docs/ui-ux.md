# UI/UX Guidelines - Skeuomorphism

## Principles
- Real-world feel
- Soft shadows
- Depth and layering
- Subtle gradients

## Components

### Article Card
- Rounded corners
- Soft shadow
- Slight inner highlight
- Hover lift effect

### Buttons
- Raised look
- Pressed state (inset shadow)

### Inputs
- Slight inset shadows
- Soft borders

## Colors
- Light backgrounds
- Neutral tones
- Avoid flat design

## Effects
- box-shadow:
  - outer: soft drop shadow
  - inner: subtle inset

Example:
box-shadow:
  6px 6px 12px #d1d1d1,
  -6px -6px 12px #ffffff;
